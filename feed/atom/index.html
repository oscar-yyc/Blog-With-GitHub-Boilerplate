Git 安装包——&gt;运行 Git 安装程序——&gt;安装完成</p> <p>如果你使用的是「Baidu」等**搜索引擎，中间的过程可能会更加繁琐曲折</p> <p>在 Windows 下大部分软件安装、配置使用各种不规范的目录，弄脏你系统的注册表，将计算机系统内部弄得脏乱不堪，这是让人难以忍受的</p> <p>由于工作、学习环境对 Windows 生态系统的依赖亦或是迁移 href="https://chocolatey.org/">Chocolatey</a></li> <li><a href="https://github.com/microsoft/winget-cli">Winget</a></li> </ul> <p>本文不做三者的比较，仅讨论 Scoop；严格来说，Scoop 不算是「包管理器」，官方解释为「Scoop 是 Windows 的命令行安装程序」，但它基本实现了 Windows 上管理软件包的流程</p> <p><strong>包管理器：</strong>或叫 <code>包管理系统</code> 是在计算机中自动安装、配置、卸载和升级软件包的工具的集合，在 <code>系统软件</code> 和 <code>应用软件</code> 的软件管理中都有大量应用，常见的一些系统/应用软件包管理器：</p> <table> git</td> </tr> <tr> <td>macOS</td> <td>Homebrew</td> <td>brew install git</td> </tr> <tr> <td>openSUSE</td> <td>zypper</td> <td>zypper install git</td> </tr> <tr> <td>Python</td> <td>Pypi</td> <td>pip <li>PowerShell 5（或更高版本，包括 PowerShell Core）</li> <li>Net Framework 4.5 及更高版本</li> <li>由于中国大陆特殊的网络情况，大概率还需要科学上网</li> </ul> <p><strong>建议环境：</strong></p> <ul> <li><a 清除所有软件包缓存 scoop cache rm * # 清除 &lt;AppName&gt; 的缓存 scoop cache rm &lt;AppName&gt; # 清除所有全局安装软件的缓存和旧版本 sudo scoop cleanup -gk * # 清除所有非全局安装软件的缓存和旧版本 scoop cleanup -k * # 删除 &lt;AppName&gt; 的旧版本 scoop cleanup &lt;AppName&gt; #
